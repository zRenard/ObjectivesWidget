# ObjectivesWidget
Garmin widget to track objectives of the year

Available on Garmin ConnectIQ

# Settings

## Event
* Name
* Date

# Display
## Glance view
* Progress bar show the year
* Tickers for the month
* Red ticker is the date of the event

## Classic view
* White circle show the year progression
* Tickers for the month
* Red ticker is the date of the event
